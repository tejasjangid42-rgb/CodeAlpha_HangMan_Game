# 🎮 Hangman Game (Python)

## 📌 Project Overview
The Hangman Game is a simple text-based Python game where the player tries to guess a randomly selected word one letter at a time. The player has a maximum of **6 incorrect guesses** before the game ends.

This project is beginner-friendly and demonstrates the use of Python fundamentals such as loops, conditionals, lists, strings, and the `random` module.

---

## 🚀 Features
- Randomly selects a word from a predefined list.
- Allows the user to guess one letter at a time.
- Displays the current progress using underscores (`_`).
- Tracks previously guessed letters.
- Allows only **6 incorrect guesses**.
- Displays a winning or losing message at the end.

## 📖 How to Play

1. Run the Python program.
2. A random word is selected.
3. Enter one letter at a time.
4. Correct guesses reveal letters in the word.
5. Wrong guesses reduce the remaining attempts.
6. Guess the complete word before your 6 attempts are over.
